# Arghyadip Portfolio React

This is a personal portfolio built with React, Vite, TypeScript, and Tailwind CSS.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```

## Features
- Responsive design
- SEO optimized
- Dark mode toggle
- Contact form

## Linting & Formatting
- ESLint and Prettier are configured. Run:
   ```bash
   npm run lint
   ```
   ```bash
   npm run format
   ```
